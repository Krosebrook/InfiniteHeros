# Feature Documentation

## 1. Dynamic Comic Generation
The core feature of the application. The AI generates a branching narrative based on user choices, complete with descriptive text and corresponding comic panel artwork. The generation process adapts to the selected genre, art style, and story tone.

## 2. Character Customization
Users can define the Hero, Sidekick (optional), and Villain (optional). Characters can be created by:
- **Uploading an Image:** The AI analyzes the image to generate a name and backstory.
- **Auto-Generation:** The AI generates a character concept and corresponding image based on the selected genre and tone.

## 3. Multiverse Map
A visual representation of the story's branching paths. Users can view their past decisions and "time travel" back to a specific node to make a different choice, exploring alternative storylines.

## 4. Interactive Elements
- **Inventory & Stats:** The application tracks the user's health, status effects, and collected items, which can influence future narrative generation.
- **Dice Roller:** Certain choices require a dice roll (e.g., D20) to determine success or failure, adding an element of chance to the story.
- **Character Chat:** Users can engage in direct conversation with the Hero, Sidekick, or Villain at any point in the story.

## 5. Accessibility & Localization
- **Text-to-Speech (TTS):** The application can read the narrative text aloud using the browser's built-in speech synthesis. Users can configure voice speed and auto-read settings.
- **Localization:** The UI is translated into 12+ languages, and the AI generates narrative content in the selected language.

## 6. Export
Users can export their generated comic as a PDF document or a ZIP file containing the individual panel images and text.
